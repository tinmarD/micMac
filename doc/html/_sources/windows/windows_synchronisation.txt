***************************
  Windows synchronisation 
*************************** 

When working with multiples windows, you can synchronise the additionnal windows with the main one. 

.. figure:: /_images/windows_sync_syncwindow_time_2.png
   :align: center

   New window with synchronisation check boxes.

There are three synchronisation options :

 - Channels 
 - Observation time
 - Time window duration
